# Turismo Tinogasta - Sitio Web (Práctica SEO y Performance)

## Consigna
Has heredado el sitio web de una agencia de turismo de Tinogasta. El cliente se queja de que no aparece en Google y que la página tarda una eternidad en cargar en el celular. Tu misión es aplicar un Hotfix de SEO y Performance.

## Tareas Técnicas (Lo que deben resolver):
1. **Fork & Clone**: Hacer un fork de este repo y clonarlo localmente.
2. **Arquitectura Semántica**:
   - Cambiar los `<div>` por `<header>`, `<main>`, `<section>` y `<footer>`.
   - Establecer la jerarquía: un solo `<h1>` con la palabra clave principal (ej: "Turismo de Aventura en Tinogasta").
3. **Optimización de Assets (SEO Técnico)**:
   - Cambiar los nombres de las fotos a algo descriptivo: `excursion-dunas-taton.jpg`.
   - Agregar el atributo `alt` a cada imagen.
   - Opcional: Convertir las imágenes a formato `.webp` para mejorar la carga.
4. **Meta-Tags**:
   - Configurar el `<title>` (máx 60 caracteres).
   - Configurar la `<meta description>` (máx 155 caracteres) invitando al clic.
5. **Git Workflow**:
   - Realizar al menos 3 commits descriptivos (ej: `feat: add semantic tags`, `fix: optimize images for SEO`).
   - Subir los cambios a su propio GitHub y enviar el link.
